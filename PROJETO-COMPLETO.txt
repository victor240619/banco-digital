🚀 COMO INICIAR

1️⃣  BACKEND:
   cd ~/banco-digital/bravus-bank
   docker-compose up -d
   (aguardar 30 segundos)

2️⃣  FRONTEND:
   cd /workspace/bravus-bank-frontend
   npm install
   npm run dev

3️⃣  ACESSAR:
   http://localhost:5173

4️⃣  LOGIN:
   Admin:   admin / admin123
   Usuário: user / user123
