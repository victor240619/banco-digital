🌐 ENDPOINTS DISPONÍVEIS
═══════════════════════════════════════════════════════════════════════════

Base URL: http://localhost:9000/api

AUTH (público):
  POST   /auth/login         → Fazer login
  POST   /auth/register      → Criar conta

USER (autenticado):
  GET    /user/profile       → Ver perfil
  GET    /user/balance       → Ver saldo
  GET    /user/transactions  → Ver transações
  POST   /user/deposit       → Depositar
  POST   /user/withdraw      → Sacar
  POST   /user/transfer      → Transferir

ADMIN (apenas admin):
  GET    /admin/dashboard              → Estatísticas
  GET    /admin/users                  → Listar usuários
  GET    /admin/users/{id}             → Detalhes usuário
  PUT    /admin/users/{id}/activate    → Ativar
  PUT    /admin/users/{id}/deactivate  → Desativar
  DELETE /admin/users/{id}             → Excluir
  GET    /admin/transactions           → Todas transações
